## Generate_Trash_Talk
User 可以選擇工程師、設計師及創業家三種職業，隨機產生幹話

## Getting Started
**Clone respository**
```
$ git clone https://github.com/Dino0403/Generate_Trash_Talk.git
```
**Install by npm**
```
$ npm install
```
$ npm run dev
```
**Browse the website on localhost**
```
http://localhost:3000/
```

## Built With
* Node.js (v10.15.0)
* Express (v4.17.1)
